<AccordionGroup>
<Accordion title="How do I modify my personal information?">
  **Modification Steps:**
  1. Click avatar in bottom left → "Settings"
  2. In the personal profile page, modify:
     - User nickname
     - User avatar
     - Identity settings (helps AI provide personalized recommendations)
     - Industry settings (helps AI provide personalized recommendations)
  3. Click "Save" button to confirm changes
  ![Modify Personal Information Steps](/images/change-info-steps.png)
</Accordion>

<Accordion title="How do I change my phone number?">
  **Change Steps:** 1. Click avatar in bottom left → "Settings" 2. In account &
  password page, modify phone number 3. Click "Change Phone" button next to
  phone number 4. Enter new phone number and get verification code 5. Enter
  verification code to complete verification 6. Save changes ![Change Phone
  Number Steps](/images/change-phone-steps.png)
</Accordion>

<Accordion title="How do I change my WeChat binding?">
  **Change Steps:** 1. Click avatar in bottom left → "Settings" 2. Select
  account & password page 3. Click "Unbind WeChat" 4. Verify with phone number
  5. Confirm successful unbinding after verification 6. Click "Bind WeChat" to
  rebind WeChat ![Change WeChat Binding Steps](/images/change-wechat-steps.png)
</Accordion>

<Accordion title="How do I bind/unbind email?">
  **Binding Steps:** 
  1. Click avatar in bottom left → "Settings" 
  2. Select account & password page 
  3. Click "Change Email" 
  4. Check email for verification code to verify identity 
  5. Confirm binding 
  6. Binding successful

**Unbinding Steps:**

1. Click avatar in bottom left → "Settings"
2. Select account & password page
3. Click "Unbind Email"
4. Verify with phone number
5. Confirm unbinding
6. Unbinding successful

![Email Binding Steps](/images/change-email-steps.png)

</Accordion>

<Accordion title="How do I change my password?">
  **Change Steps:** 1. Click avatar in bottom left → "Settings" 2. Select
  account & password page 3. Click "Change Password" 4. Enter current password
  5. Enter new password (twice for confirmation) 6. Click "Confirm" to save new
  password ![Change Password Steps](/images/change-password-steps.png)
</Accordion>

<Accordion title="How do I set my avatar?">
  **Setting Steps:** 1. Click avatar in bottom left → "Settings" 2. In personal
  profile page, click current avatar 3. Choose from: - Upload new image - Select
  from gallery - Use default avatars 4. Adjust image size and position 5. Click
  "Confirm" to save ![Set Avatar Steps](/images/set-avatar-steps.png)
</Accordion>

<Accordion title="How do I set AI personalization?">
  **Setting Steps:**
  1. Click avatar in bottom left → "Settings"
  2. In personal profile page, find "AI Personalization"
  3. Set your preferences:
     - Professional field (hardware, software, etc.)
     - Interest areas (AI, IoT, automotive, etc.)
     - Experience level (beginner, intermediate, expert)
     - Preferred component types
  4. Click "Save" to apply settings
  
  <Info>
  These settings help AI provide more relevant search results and recommendations
  </Info>
  
  ![AI Personalization Steps](/images/ai-personalization-steps.png)
</Accordion>
</AccordionGroup>

### Two-Factor Authentication

<Steps>
<Step title="Enable 2FA">
  Go to Settings → Security → Two-Factor Authentication
</Step>

<Step title="Choose Method">
  Select your preferred 2FA method: - SMS verification - Email verification -
  Authenticator app
</Step>

<Step title="Complete Setup">
  Follow the setup instructions and test the verification
</Step>
</Steps>

## Troubleshooting

### Common Issues

<AccordionGroup>
<Accordion title="Can't Access Account">
If you can't access your account:
1. Check if you're using the correct email/phone
2. Try resetting your password
3. Check if your account is locked
4. Contact support for assistance
</Accordion>

<Accordion title="Verification Code Issues">
  If verification codes aren't working: 1. Check if the code has expired 2.
  Ensure you're entering the correct code 3. Check your spam folder for emails
  4. Request a new code if needed
</Accordion>

<Accordion title="Profile Update Problems">
If profile updates aren't saving:
1. Check if all required fields are filled
2. Ensure file sizes are within limits
3. Try refreshing the page
4. Clear browser cache and try again
</Accordion>
</AccordionGroup>
