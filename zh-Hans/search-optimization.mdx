## 概述

PartGenie 的 AI 搜索功能强大，但掌握正确的搜索技巧可以显著提高搜索效率和结果准确性。本指南将教您如何优化搜索策略，快速找到所需的元器件。

## 搜索策略框架

### 1. 关键词层次化

**第一层：核心器件类型**

- 电阻、电容、电感
- 晶体管、二极管、IC
- 连接器、开关、传感器

**第二层：功能分类**

- 电源管理：稳压器、转换器、保护器件
- 信号处理：放大器、滤波器、ADC/DAC
- 数字逻辑：微控制器、存储器、逻辑门

**第三层：具体参数**

- 电气参数：电压、电流、功率
- 物理参数：封装、尺寸、引脚数
- 性能参数：精度、速度、功耗

### 2. 搜索语法技巧

#### 使用引号精确匹配

```
"LM358" - 精确搜索LM358
"运算放大器" - 精确匹配中文术语
```

#### 使用布尔运算符

```
(运算放大器 OR 运放) AND (单电源 OR 5V)
MOSFET AND (N沟道 OR N-channel) AND TO-220
```

#### 排除不需要的结果

```
微控制器 -Arduino -ESP32
电源管理 -充电器
```

## 常见搜索错误及避免

### ❌ 错误示例

**过于宽泛：**

```
电子元件
```

**参数不匹配：**

```
12V 5A 电阻（电阻不能提供5A电流）
```

**术语不准确：**

```
小芯片（应该使用具体封装名称）
```

**需求矛盾：**

```
低功耗 高频 大电流（这些要求通常矛盾）
```

### ✅ 正确示例

**具体明确：**

```
N沟道MOSFET 30V 5A TO-220封装
```

**参数合理：**

```
限流电阻 12V 20mA LED保护
```

**术语准确：**

```
SOT-23封装 运算放大器 单电源
```

**需求平衡：**

```
低功耗运算放大器 1MHz带宽 1mA静态电流
```
