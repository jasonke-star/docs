## 搜索技巧详解

### 基础搜索技巧

<Steps>
<Step title="使用自然语言">
  **推荐方式：**
  
  ```
  ✅ 好的搜索：
  - "需要一颗 3.3V 的 LDO 稳压器"
  - "Arduino 项目用的温湿度传感器"
  - "电机驱动用的 H 桥芯片"
  ```
  
  ```
  ❌ 避免的搜索：
  - "稳压器"
  - "传感器"
  - "芯片"
  ```
</Step>

<Step title="包含关键参数">
  **重要参数：**
  - **电压/电流**：工作电压、最大电流
  - **封装**：封装类型和尺寸
  - **接口**：通信接口类型
  - **精度**：精度要求
  - **温度**：工作温度范围
</Step>

<Step title="描述应用场景">
  **应用描述：**
  
  ```
  示例：
  - "用于手机充电器的电源管理芯片"
  - "汽车电子用的 CAN 收发器"
  - "工业控制用的隔离放大器"
  ```
</Step>
</Steps>

## 常见问题

<AccordionGroup>
<Accordion title="搜索结果太少怎么办？">
  **解决方案：**
  - 使用更通用的关键词
  - 放宽参数要求
  - 尝试同义词搜索
  - 检查拼写是否正确

<Accordion title="搜索结果太多怎么办？">
  **解决方案：**
  - 添加更多限制条件
  - 使用筛选器缩小范围

  - 指定具体的应用场景
</Accordion>
