## 如何使用 AI 搜索

<Steps>
<Step title="打开搜索页面">
  登录 PartGenie 后，您会看到主搜索界面
  
  <Info>
  搜索栏位于页面中间，带有"输入元器件型号、参数或您想要解决的问题..."提示
  </Info>
</Step>

<Step title="输入搜索需求">
  在搜索栏中输入您的需求，可以使用自然语言： **简单描述：** ``` "需要一颗 3.3V
  的稳压器" ``` **详细描述：** ``` "LDO 稳压器，输入 5V，输出 3.3V，电流
  1A，SOT-23 封装" ``` **应用场景：** ``` "用于 Arduino 项目的电源管理芯片" ```
</Step>

<Step title="查看搜索结果">
  AI 会智能分析您的需求并返回匹配的元器件列表
  <Tip>结果按相关度排序，最匹配的产品会以卡片的方式显示在概览标签页上方</Tip>
</Step>

<Step title="筛选和优化">
  你可以在元器件标签页使用筛选器进一步调整搜索范围：
  - **工作电压**：选择适合的电压范围
  - **封装类型**：选择适合的封装
</Step>
</Steps>

## 搜索策略

### 搜索类型

<Table>
  | 搜索类型 | 关键词策略 | 示例 | |----------|------------|------| | 功能搜索 |
  使用功能描述词 | "稳压器"、"放大器"、"转换器" | | 参数搜索 | 包含具体数值 |
  "3.3V"、"1A"、"100MHz" | | 应用搜索 | 描述使用场景 | "Arduino"、"汽车"、"工业"
  | | 品牌搜索 | 指定制造商 | "ST"、"TI"、"ADI" |
</Table>

### 搜索技巧

<Steps>
<Step title="从宽到窄">
  1. 先进行宽泛搜索
  2. 使用筛选器缩小范围
  3. 逐步细化搜索条件
</Step>

<Step title="多角度搜索">
  1. 从功能角度搜索 2. 从参数角度搜索 3. 从应用角度搜索 4. 综合多个角度
</Step>

<Step title="学习搜索结果">
  1. 分析搜索结果的相关性
  2. 调整搜索关键词
  3. 优化搜索策略
</Step>
</Steps>

## 数据导出

Pro 用户可以将搜索结果导出为 Excel 文件：

<Steps>
<Step title="选择导出内容">
  在搜索结果页面，选择要导出的元器件
</Step>

<Step title="点击导出按钮">点击"导出"按钮，选择导出格式</Step>

<Step title="下载文件">
  系统会生成包含详细信息的 Excel 文件供您下载
</Step>
</Steps>

## 常见问题

<AccordionGroup>
<Accordion title="搜索无结果怎么办？">
  可能是平台数据库暂无收录相关元器件数据，您可以：
  
  1. 尝试使用更通用的关键词
  2. 调整搜索条件
  3. 申请数据收录（联系客服）
</Accordion>

<Accordion title="如何提高搜索准确性？">
  1. 使用更具体的关键词 2. 结合参数和应用场景描述 3. 参考"最佳实践"中的搜索技巧
</Accordion>

<Accordion title="搜索结果不准确怎么办？">
  1. 检查关键词是否准确
  2. 尝试不同的搜索方式
  3. 使用筛选器进一步缩小范围
</Accordion>
</AccordionGroup>
