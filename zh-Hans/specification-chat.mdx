## 如何使用规格书对话

<Steps>
<Step title="查看产品">
  在搜索结果中点击产品卡片
</Step>

<Step title="找到对话功能">在产品详情页找到"规格书对话"按钮</Step>

<Step title="开始对话">
  点击按钮进入对话界面
</Step>
</Steps>

## 实际应用案例

### 案例 1：参数确认

<Steps>
<Step title="确认基本参数">
  ```
  问题："这个 STM32F103C8T6 的基本参数是什么？"
  回答：工作电压 2.0V-3.6V，主频 72MHz，Flash 64KB，RAM 20KB
  ```
</Step>

<Step title="确认工作条件">
  ``` 问题："在什么温度范围内可以正常工作？" 回答：工作温度 -40°C 到
  +85°C，存储温度 -65°C 到 +150°C ```
</Step>

<Step title="确认接口特性">
  ```
  问题："支持哪些通信接口？"
  回答：支持 UART、SPI、I2C、CAN、USB 等接口
  ```
</Step>
</Steps>

### 案例 2：应用咨询

<Steps>
<Step title="描述应用场景">
  ```
  问题："我想用这个芯片做 Arduino 项目，合适吗？"
  回答：这个芯片完全兼容 Arduino 生态系统，非常适合 Arduino 项目开发
  ```
</Step>

<Step title="询问设计要点">
  ``` 问题："设计时需要注意什么？"
  回答：需要注意电源设计、时钟配置、引脚分配等关键点 ```
</Step>

<Step title="获取建议">
  ```
  问题："有什么推荐的开发工具？"
  回答：推荐使用 Arduino IDE、PlatformIO 或 Keil MDK 等开发环境
  ```
</Step>
</Steps>
