## 常见问题概览

这里收集了用户最常遇到的问题和解答，帮助您快速解决使用中遇到的问题。

## 账户和登录

### 账户注册

<AccordionGroup>
<Accordion title="如何注册 PartGenie 账户？">
  **注册步骤：**
  1. 访问 [partgenie.ai](https://partgenie.ai)
  2. 点击"注册"按钮
  3. 填写邮箱和密码
  4. 验证邮箱地址
  5. 完善个人资料
  
  **支持方式：**
  - 邮箱注册
  - Google 账户登录
  - GitHub 账户登录
  - 企业邮箱注册
</Accordion>

<Accordion title="注册时收不到验证邮件怎么办？">
  **可能原因和解决方案：** - 检查垃圾邮件文件夹 - 确认邮箱地址正确 - 等待 5-10
  分钟 - 重新发送验证邮件 - 联系客服：support@partgenie.ai
</Accordion>

<Accordion title="可以使用企业邮箱注册吗？">
  **可以，企业邮箱注册的优势：**
  - 自动识别企业用户
  - 享受企业级功能
  - 团队协作权限
  - 专属技术支持
  
  **注册时选择企业邮箱即可自动识别。**
</Accordion>
</AccordionGroup>

### 登录问题

<AccordionGroup>
<Accordion title="忘记密码怎么办？">
  **重置密码步骤：**
  1. 点击登录页面的"忘记密码"
  2. 输入注册邮箱
  3. 查收重置邮件
  4. 点击邮件中的重置链接
  5. 设置新密码
  
  **如果收不到邮件，请联系客服。**
</Accordion>

<Accordion title="账户被锁定怎么办？">
  **账户锁定的常见原因：** - 多次输入错误密码 - 异常登录行为 - 违反服务条款
  **解锁方法：** - 等待 30 分钟自动解锁 - 通过邮箱重置密码 - 联系客服申请解锁
</Accordion>

<Accordion title="如何启用两步验证？">
  **启用步骤：**
  1. 登录账户 → 账户设置
  2. 安全设置 → 两步验证
  3. 选择验证方式（短信/应用）
  4. 按照提示完成设置
  
  **推荐使用身份验证器应用，更安全。**
</Accordion>
</AccordionGroup>

## 搜索和功能

### AI 智能搜索

<AccordionGroup>
<Accordion title="搜索不到想要的产品怎么办？">
  **优化搜索的方法：**
  - 使用更具体的关键词
  - 尝试不同的描述方式
  - 使用筛选器缩小范围
  - 检查拼写是否正确
  - 尝试同义词搜索
  
  **示例：**
  - ❌ "芯片"
  - ✅ "STM32F103C8T6 微控制器"
  - ✅ "3.3V LDO 稳压器 1A"
</Accordion>

<Accordion title="搜索结果太多怎么办？">
  **缩小搜索范围：** - 添加更多限制条件 - 使用高级筛选器 - 指定具体的应用场景 -
  选择特定的供应商 - 设置价格范围 **筛选器选项：** - 工作电压范围 - 封装类型 -
  供应商品牌 - 价格区间 - 库存状态
</Accordion>

<Accordion title="如何提高搜索准确性？">
  **搜索技巧：**
  - 使用自然语言描述需求
  - 包含关键参数信息
  - 描述应用场景
  - 使用专业术语
  - 结合多个关键词
  
  **示例：**
  - "用于 Arduino 项目的 3.3V 稳压器，1A 输出，SOT-23 封装"
</Accordion>
</AccordionGroup>

### BOM 管理

<AccordionGroup>
<Accordion title="如何创建 BOM？">
  **创建步骤：**
  1. 点击"BOM 管理"
  2. 点击"新建 BOM"
  3. 填写 BOM 基本信息
  4. 添加元器件
  5. 设置数量和参数
  6. 保存 BOM
  
  **添加元器件的方法：**
  - 搜索添加
  - 手动添加
  - 导入文件
  - 从收藏添加
</Accordion>

<Accordion title="如何批量修改 BOM 中的元器件数量？">
  **批量修改步骤：** 1. 在 BOM 页面选择多个元器件 2. 点击"批量编辑" 3.
  选择"修改数量" 4. 输入新的数量 5. 确认修改 **支持的操作：** - 批量修改数量 -
  批量修改备注 - 批量删除元器件 - 批量移动到其他 BOM
</Accordion>

<Accordion title="BOM 可以分享给其他人吗？">
  **分享方式：**
  - 邀请团队成员协作
  - 生成分享链接
  - 导出为文件分享
  - 设置访问权限
  
  **权限级别：**
  - 查看权限：只能查看
  - 编辑权限：可以修改
  - 管理权限：完全管理
</Accordion>
</AccordionGroup>

## 订阅和计费

### 会员计划

<AccordionGroup>
<Accordion title="免费版有什么限制？">
  **免费版限制：**
  - AI 搜索：10 次/天
  - BOM 管理：1 个 BOM
  - 产品对比：3 个产品
  - 收藏功能：50 个产品
  - 团队协作：不支持
  
  **升级建议：**
  如果经常使用，建议升级到专业版。
</Accordion>

<Accordion title="如何升级会员计划？">
  **升级步骤：** 1. 进入账户设置 2. 选择"订阅管理" 3. 选择要升级的计划 4.
  确认升级费用 5. 完成支付 **升级优势：** - 立即生效 - 按比例计费 - 享受新功能 -
  无数据丢失
</Accordion>

<Accordion title="可以随时取消订阅吗？">
  **取消政策：**
  - 可以随时取消
  - 下个计费周期生效
  - 已付费期间正常使用
  - 支持 30 天退款
  
  **取消步骤：**
  1. 账户设置 → 订阅管理
  2. 选择"取消订阅"
  3. 确认取消原因
  4. 完成取消操作
</Accordion>
</AccordionGroup>

### 支付和发票

<AccordionGroup>
<Accordion title="支持哪些支付方式？">
  **支付方式：**
  - 支付宝
  - 微信支付
  - 银联卡
  - 信用卡（Visa、MasterCard）
  - 企业银行转账
  
  **支付安全：**
  - SSL 加密传输
  - PCI DSS 认证
  - 不存储卡号信息
</Accordion>

<Accordion title="如何申请发票？">
  **申请步骤：** 1. 支付完成后申请发票 2. 选择发票类型（个人/企业） 3.
  填写发票信息 4. 提交申请 5. 下载电子发票 **发票类型：** - 个人发票 -
  企业发票（需要税号） - 电子发票 - 纸质发票（邮寄）
</Accordion>

<Accordion title="如何申请退款？">
  **退款条件：**
  - 30 天内可申请退款
  - 未使用高级功能
  - 无违规行为
  
  **申请步骤：**
  1. 联系客服申请退款
  2. 提供退款原因
  3. 等待审核（1-3 工作日）
  4. 原路退回费用（3-7 工作日）
</Accordion>
</AccordionGroup>
