## 标题

最适合用作节标题。

```md
## 标题
```

### 副标题

最适合用作子节标题。

```md
### 副标题
```

<Tip>

每个**标题**和**副标题**都会创建一个锚点，并显示在右侧的目录中。

</Tip>

## 文本格式

我们支持大多数 markdown 格式。只需在文本周围添加 `**`、`_` 或 `~` 来格式化它。

| 样式   | 如何编写   | 结果     |
| ------ | ---------- | -------- |
| 粗体   | `**粗体**` | **粗体** |
| 斜体   | `_斜体_`   | _斜体_   |
| 删除线 | `~删除线~` | ~删除线~ |

您可以组合这些。例如，编写 `**_粗体和斜体_**` 来获得 **_粗体和斜体_** 文本。

您需要使用 HTML 来编写上标和下标文本。也就是说，在文本周围添加 `<sup>` 或 `<sub>`。

| 文本大小 | 如何编写          | 结果            |
| -------- | ----------------- | --------------- |
| 上标     | `<sup>上标</sup>` | <sup>上标</sup> |
| 下标     | `<sub>下标</sub>` | <sub>下标</sub> |

## 链接到页面

您可以通过将文本包装在 `[]()` 中来添加链接。您会编写 `[链接到谷歌](https://google.com)` 来[链接到谷歌](https://google.com)。

链接到您文档中的页面需要是根相对的。基本上，您应该包含整个文件夹路径。例如，`[链接到文本](/writing-content/text)` 链接到我们组件部分中的"文本"页面。

像 `[链接到文本](../text)` 这样的相对链接会打开得更慢，因为我们无法轻易优化它们。

## 引用块

### 单行

要创建引用块，请在段落前添加 `>`。

> 多萝西跟随她穿过城堡中许多美丽的房间。

```md
> 多萝西跟随她穿过城堡中许多美丽的房间。
```

### 多行

> 多萝西跟随她穿过城堡中许多美丽的房间。
>
> 女巫吩咐她清洗锅碗瓢盆，扫地，并保持火炉燃烧。

```md
> 多萝西跟随她穿过城堡中许多美丽的房间。
>
> 女巫吩咐她清洗锅碗瓢盆，扫地，并保持火炉燃烧。
```

### LaTeX

Mintlify 通过 Latex 组件支持 [LaTeX](https://www.latex-project.org)。

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
