<AccordionGroup>
<Accordion title="我如何修改个人信息？">
  **修改步骤：**
  1. 点击左下角头像 → "设置"
  2. 在个人资料页面修改：
     - 用户昵称
     - 用户头像
     - 身份设置（将有助于AI推荐个性化结果）
     - 行业设置（将有助于AI推荐个性化结果）
  3. 点击"保存"按钮确认更改
  ![修改个人信息步骤](/images/change-info-steps.png)
</Accordion>

<Accordion title="我如何更换手机号？">
  **更换步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 在账号与密码页面修改手机号 
  3. 点击手机号旁的"更换手机"按钮 
  4. 输入新手机号并获取验证码 
  5. 输入验证码完成验证
  6. 保存更改
  
  ![更换手机号步骤](/images/change-phone-steps.png)
</Accordion>

<Accordion title="我如何更换关联微信？">
  **更换步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 选择账号与密码页面 
  3. 点击"解绑微信" 
  4. 进行手机号验证 
  5. 验证通过后确认解绑成功 
  6. 点击"绑定微信"重新绑定微信
  
  ![更换微信绑定步骤](/images/change-wechat-steps.png)
</Accordion>

<Accordion title="我如何解绑邮箱？">
  **绑定步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 选择账号与密码页面 
  3. 点击"更换邮箱" 
  4. 查收邮箱验证码进行身份验证 
  5. 确认解绑 
  6. 解绑成功
  
  ![解绑邮箱步骤](/images/unbind-email-steps.png)
</Accordion>

<Accordion title="我如何修改密码？">
  **修改步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 选择账号与密码页面 
  3. 点击"设置密码" 
  4. 输入当前密码进行验证 
  5. 输入新密码并确认 
  6. 保存设置
  
  ![修改密码步骤](/images/change-password-steps.png)
  **密码安全要求：** - 密码长度6-24位 - 至少包含字母、数字、符号2种组合
</Accordion>

<Accordion title="我如何绑定微信？">
  **绑定步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 选择账号与密码页面 
  3. 点击"绑定微信" 
  4. 使用微信扫码授权 
  5. 确认绑定成功 
  
  ![绑定微信步骤](/images/bind-wechat-steps.png)
  
  **注意事项：** 
  - 一个微信只能绑定一个账户 
  - 绑定后可使用微信快速登录 
  - 如需解绑，请参考解绑操作
</Accordion>

<Accordion title="我如何绑定邮箱？">
  **绑定步骤：** 
  1. 点击左下角头像 → "设置" 
  2. 选择账号与密码页面    
  3. 点击"绑定邮箱"  
  4. 输入邮箱并验证邮箱 
  5. 查收邮件并点击验证链接
  6. 验证成功后邮箱绑定完成
  
  ![绑定邮箱步骤](/images/bind-email-steps.png)

  **注意事项：**

  - 如果未收到邮件，请检查垃圾邮件文件夹
  - 一个邮箱只能绑定一个账户
  - 绑定邮箱后可使用邮箱快速登录
  </Accordion>
