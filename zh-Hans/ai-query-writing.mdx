## 概述

在 PartGenie 中，撰写高质量的查询是获得精准元器件推荐的关键。本指南将教您如何构建 AI 能够准确理解和处理的查询语句。

## 查询结构要素

### 1. 明确指定元器件类型

**❌ 模糊查询：**

```
我需要一个电子元件
```

**✅ 精准查询：**

```
我需要一个用于电源管理的MOSFET晶体管，N沟道，TO-220封装
```

### 2. 包含关键电气参数

**必需参数：**

- 电压等级（如：12V, 24V, 400V）
- 电流容量（如：1A, 5A, 20A）
- 功率等级（如：1W, 10W, 100W）

**示例：**

```
寻找一个12V输入、5V输出、3A电流的DC-DC降压转换器
```

### 3. 指定应用场景

**❌ 通用描述：**

```
用于电路的电阻
```

**✅ 具体应用：**

```
用于LED限流保护的电阻，12V电源驱动3.3V LED
```

## 高级查询技巧

### 使用技术术语

- 使用标准的电子工程术语
- 包含封装类型（SOT-23, QFN, BGA 等）
- 指定工作温度范围
- 提及特殊要求（如：低噪声、高频、低功耗）

### 示例对比

**基础查询：**

```
我需要一个放大器
```

**优化查询：**

```
我需要一个运算放大器，单电源供电5V，带宽至少10MHz，用于音频信号放大，SOT-23封装，工作温度-40°C到+85°C
```

## 常见查询模式

### 1. 替换查询

```
替换型号：LM358，需要相同功能的运算放大器，但要求更低的功耗
```

### 2. 升级查询

```
升级现有方案：当前使用74HC04，需要更快的逻辑门，保持相同的引脚兼容性
```

### 3. 成本优化查询

```
寻找LM7805的替代方案，要求更低的成本，保持相同的5V稳压功能
```

### 4. 性能提升查询

```
提升现有设计：当前使用普通二极管，需要更快的开关二极管用于高频应用
```

## 查询优化建议

### 1. 分步骤描述

如果需求复杂，可以分步骤描述：

```
第一步：我需要一个微控制器，ARM Cortex-M4内核
第二步：要求至少64KB Flash，32KB RAM
第三步：需要内置ADC和PWM功能
第四步：封装要求LQFP64
```

### 2. 使用比较和排除

```
寻找类似STM32F4系列的微控制器，但要求更低的功耗，排除需要外部晶振的型号
```

### 3. 包含设计约束

```
在PCB尺寸限制下，寻找最小的DC-DC转换器，输入12V，输出3.3V，电流1A
```

## 避免的常见错误

### ❌ 过于宽泛

```
我需要电子元件
```

### ❌ 缺少关键参数

```
我需要一个电容（没有指定容量、电压、类型）
```

### ❌ 使用非标准术语

```
我需要一个"小芯片"（应该使用具体的封装名称）
```

### ❌ 混合多个不相关需求

```
我需要一个电阻和一个电容和一个电感（应该分别查询）
```

## 查询模板

### 基础模板

```
[元器件类型] + [关键参数] + [应用场景] + [特殊要求]
```

### 替换查询模板

```
替换 [原型号]，要求 [新需求]，保持 [兼容性要求]
```

### 升级查询模板

```
升级 [当前方案]，提升 [性能指标]，保持 [现有约束]
```

## 实践练习

尝试使用以下场景练习查询撰写：

1. **电源管理**：设计一个 12V 转 5V 的电源模块
2. **信号处理**：构建一个音频前置放大器
3. **数字控制**：选择微控制器用于电机控制
4. **传感器接口**：连接温度传感器到微控制器

## 总结

高质量的查询是获得精准元器件推荐的基础。通过明确指定元器件类型、电气参数、应用场景和特殊要求，您可以让 AI 更好地理解您的需求，从而获得更准确、更相关的元器件推荐。

记住：**越具体，越精准！**
